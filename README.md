#  VisionSketch AI

###  AI-Powered Virtual Whiteboard Using Hand Tracking

VisionSketch AI is a computer vision-based virtual drawing board that
allows users to draw in the air using hand gestures captured through a 
webcam. It uses **OpenCV** and **MediaPipe** to track hand landmarks and convert
finger movements into real-time drawing on screen.

---

##  Features

-  Real-time hand tracking using webcam
-  Draw on screen using index finger gestures
-  Multiple color support
-  AI-based gesture recognition (MediaPipe)
-  Lightweight and fast (no heavy GPU required)
-  Clear and reset drawing canvas
-  Modular project structure for easy extension

---

##  Tech Stack

- Python 🐍
- OpenCV 👁️
- MediaPipe 🤖
- NumPy 📊
